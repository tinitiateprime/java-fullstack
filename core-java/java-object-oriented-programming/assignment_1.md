# Java OOP Concepts — MCQ Assignment

    A) Object Oriented Programming  
    B) Only Object Programming  
    C) Organized Object Protocol  
    D) Oriented Object Process

    C) Compilation  
    D) Polymorphism

    A) Faster compilation  
    B) Code reusability and modularity

    A) Inheritance  
    B) Encapsulation

    D) Abstraction

    A) Inheritance

    D) Polymorphism

    B) inherits  
    C) extends  
    D) super

    A) Single  
    B) Multilevel

    B) Using many variables

    D) Data hiding

    A) Compile-time polymorphism  
    B) Runtime polymorphism  
    C) Dynamic polymorphism

    A) Compile-time  
    B) Runtime  
    C) Static binding  
    D) Early binding

    A) typeOf  
    B) instanceOf  
    C) instanceof  
    D) is

    A) Inheritance  
    B) Encapsulation  
    C) Abstraction  
    D) Polymorphism

    A) Same method name, same parameters in subclass  
    B) Same method name, different parameters  
    C) Same name, different classes without relation

    A) final methods  
    B) static methods  
    C) private methods  
    D) All of the above

    B) Abstract class may have both abstract and concrete methods

    D) Abstract methods must be static

    A) Yes  
    B) No  
    C) Only with super()  
    D) Only if it has constructor

    A) virtual  
    B) final  
    C) abstract

    A) inherit  
    B) extends  
    C) implements  
    D) abstract  

22. What is an interface in Java?

    A) A class with all private methods  
    B) A class that defines constants only  
    C) A collection of abstract methods  
    D) A subclass of Object

23. What type of inheritance is supported by interfaces?

    A) Multiple inheritance  
    B) Single inheritance  
    C) Multilevel inheritance  
    D) None

    A) The parent class object  
    B) The current class instance  
    C) The interface reference  
    D) The static context

    A) To call the subclass constructor  
    B) To call the parent class constructor  
    C) To create new instance  
    D) To override a method

    A) Static binding  
    B) Virtual method invocation (runtime)  
    C) Compile-time linking  
    D) Function overloading

    A) Abstract classes support multiple inheritance  
    B) Interfaces cannot have methods  
    C) Both can have abstract methods  
    D) Interfaces can have constructors

    A) protected  
    B) private  
    C) default  
    D) public

29. What does encapsulation mainly protect?

    A) Methods  
    B) Variables (data)  
    C) Packages  
    D) Classes

    A) Car with private engine and public start/stop methods  
    B) Inheriting methods from parent  
    C) Overriding methods  
    D) Using abstract classes

    A) default and static methods  
    B) private methods  
    C) only abstract methods  
    D) final methods

    A) Calling one constructor from another using this() or super()  
    B) Calling multiple methods

    D) Finalizing constructors

    A) Polymorphism  
    B) Inheritance  
    C) Abstraction  
    D) Encapsulation

    B) Method overriding

    B) Overriding methods

    A) Inheritance  
    B) Encapsulation  
    C) Abstraction  
    D) Polymorphism

    A) Static binding  
    B) Late binding  
    C) Runtime binding

    B) Constructors are not inherited but can be called using super()
