# Java Abstraction — MCQ Assignment
1. What is abstraction in Java?

    A) Hiding internal details and showing only essential features  
    B) Making all methods private  
    C) Showing complete implementation  
    D) Preventing inheritance

2. Which keyword is used to declare an abstract class?

    A) class  
    B) abstract  
    C) static  
    D) interface

3. Which of the following is true for abstract classes?

    A) They can have both abstract and concrete methods  
    B) They cannot have constructors  
    C) They must be declared final  
    D) They can’t have variables

4. Can we create an object of an abstract class?

    A) Yes, directly  
    B) No, but through subclass instance  
    C) Yes, using new keyword  
    D) Only if it has constructors

5. What is an abstract method?

    A) Method with no body, meant to be implemented by subclasses  
    B) Method that is always private  
    C) Method that cannot be inherited  
    D) Method defined with final

6. What happens if a subclass does not implement all abstract methods of its superclass?

    A) Compilation error  
    B) Runtime error  
    C) It becomes an abstract class itself  
    D) It executes default methods

7. Which of the following can have constructors?

    A) Abstract classes  
    B) Interfaces  
    C) Both  
    D) None

8. Which of the following can have both abstract and non-abstract methods?

    A) Interface   
    B) Abstract class  
    C) Enum  
    D) Final class

9. Which keyword is used by a subclass to provide implementation for an interface?

    A) extends  
    B) inherit  
    C) implements  
    D) super

10. Can an interface extend another interface?

    A) Yes  
    B) No  
    C) Only if abstract  
    D) Only in Java 8

11. What is multiple inheritance in Java?

    A) A class implementing multiple interfaces  
    B) A class extending multiple classes  
    C) A class extending an abstract class and interface together  
    D) Not allowed in any form

12. Which of these is not allowed inside an interface?

    A) abstract methods  
    B) static methods  
    C) constructors  
    D) constants

13. What kind of methods can be declared in interfaces (Java 8+)?

    A) abstract, default, and static  
    B) final only  
    C) private and static only  
    D) abstract only

14. What is a functional interface?

    A) An interface with exactly one abstract method  
    B) An interface with only static methods  
    C) An interface with more than one abstract method  
    D) Any interface extending another

15. Which annotation ensures an interface is functional?

    A) @Functional  
    B) @SingleAbstract  
    C) @FunctionalInterface  
    D) @Interface

16. Can an interface have variables?

    A) Yes, they are implicitly public, static, and final  
    B) Yes, but only private  
    C) No, interfaces cannot have variables  
    D) Yes, but they are mutable  

17. Which statement about abstract classes and interfaces is true?

    A) Abstract classes can have state (fields), interfaces cannot  
    B) Interfaces can have constructors  
    C) Abstract classes cannot have methods  
    D) Both are instantiated directly  

18. Which method type in an abstract class must be implemented by subclasses?

    A) abstract  
    B) static  
    C) private  
    D) final  

19. What is the difference between abstract and concrete methods?

    A) Abstract has no body; concrete has implementation  
    B) Concrete methods are always static  
    C) Abstract methods are final  
    D) Both require implementation  

20. Why use abstraction in Java?

    A) To hide implementation details and increase flexibility  
    B) To improve runtime speed  
    C) To avoid inheritance  
    D) To allow direct object creation
